# shop-shop
Una tienda desde la Comunidad.

Solución creada con el objetivo de implementar funcionalidades básicas de una tienda virtual. 

- Operaciones de Clientes.
- Operaciones de Productos.
- Operaciones de Venta.

NOTA: Este proyecto tiene como objetivo fundamental, validar competencias para el desarrollo backend en el stack Java.


Tecnologías:
- Lenguaje: Java.
- Framework: Spring, Spring Boot.
- BD: Postgres.
- Arquitectura: Orientada a servicios.
- Pruebas:JUnit, Mockito.
